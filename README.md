#  Hôtel Lumière – Luxury 5-Star Hotel Website  
A modern, elegant, and fully responsive 5-star hotel booking website built with **HTML, Tailwind CSS, and JavaScript**.  
This project focuses on luxury UI, smooth animations, and real-world hotel booking flow.

##  Features  
###  **Modern & Premium UI**
- Luxury 5-star hotel theme  
- Golden & dark premium color palette  
- Hero section with video/image background  

###  **Booking System UI**
- Check-in date & Check-out date inputs  
- Guests selector  
- “Check Availability” booking CTA  

###  **Fully Responsive**
- Mobile-first layout  
- Smooth hamburger navigation  
- Adaptive booking card for all screens  

###  **Pages Included**
- Home  
- Rooms  
- Gallery  
- Dining  
- Facilities  
- About  
- Contact  


##  Tech Stack  
- **HTML5**  
- **Tailwind CSS**  
- **JavaScript (Vanilla)**  
- **Responsive Design (Mobile-First)**
